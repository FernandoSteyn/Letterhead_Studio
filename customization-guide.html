üìù Fonts & Typography

  • Playfair Display: Elegant serif for company names and headings
  • Roboto: Monospace for registration numbers and technical information

üî§ Changing Fonts

To use different fonts, update the Google Fonts import and CSS:

<!-- In HTML head section --> <link href="https://fonts.googleapis.com/css2?family=YourFont:wght@300;400;600;700&display=swap" rel="stylesheet">
/* In CSS file */ body { font-family: 'Your Font', sans-serif; } .company-title { font-family: 'Your Heading Font', serif; }
üí° Font Selection Tips

Choose professional fonts that maintain readability at different sizes. Sans-serif fonts generally work better for digital viewing, while serif fonts can enhance printed materials.

üìê Page Setup

For best print results, configure your browser's print settings:

  1. Paper Size

    Set to A4 (210mm √ó 297mm) for international standard or Letter (8.5" √ó 11") for US standard.

  2. Margins

    Use minimum margins (usually 0.5" or 12.7mm) for maximum content area.

  3. Background Graphics

    Enable "Print background graphics" to include colors and gradients.

  4. Scale

    Set scale to 100% to maintain proper proportions and sizing.

üîß Troubleshooting

üêõ Common Issues

Fonts Not Loading

Solution: Ensure internet connection for Google Fonts, or download fonts locally and update CSS imports.

3D Effects Not Working

Solution: Update your browser to the latest version. 3D effects require modern CSS support.

Print Colors Missing

Solution: Enable "Print background graphics" in browser print settings and check printer color settings.

Mobile Display Issues

Solution: The templates are responsive. Ensure viewport meta tag is present in HTML head section.

Assets Not Loading

Solution: Check file paths and ensure all assets are in the correct folders relative to HTML files.
